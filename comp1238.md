## Week 2 - Bits and Bytes
- A bit is 0 or 1, and 1 byte = 8 bits.
- Binary numbers use base-2 and are how computers store data.
- RAM is volatile memory (data lost when power off), SSD/HDD are non-volatile.
- ASCII table maps numbers to text characters (example: 65 = A).
